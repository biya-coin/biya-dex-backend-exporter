使用说明

  • 获取正确的面板 ID：在 Grafana 中点击面板右上角菜单 → "Share" → "Embed"，复制 iframe URL 中的 panelId 参数值
  • 注意：Grafana 导入仪表板时可能会重新分配面板 ID，所以配置文件中的 ID（1-8）可能与实际 ID 不同
  • URL 格式：使用 d-solo 路径和 panelId 参数（这是 Grafana 推荐的单个面板嵌入方式）
  • 时间范围：可以通过修改 from 和 to 参数调整（当前为 now-1h 到 now）
  • 刷新间隔：可以通过 refresh 参数调整（当前为 10 秒）
  • 示例：Gas 价格面板的实际 ID 是 11，所以使用 panelId=11
